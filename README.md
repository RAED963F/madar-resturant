# 🌟 مدار - موقع مطعم عربي فاخر في موسكو

## نظرة عامة
موقع إلكتروني متكامل ومتطور لمطعم "مدار" - أكبر وأفخم مطعم عربي في موسكو وروسيا. الموقع يجمع بين الأصالة العربية والتصميم الحديث مع تأثيرات بصرية متقدمة.

## ✨ المميزات الرئيسية

### 🎨 التصميم
- **تصميم عربي أصيل** مع لمسات فاخرة باللون الذهبي
- **خطوط عربية احترافية**: Cairo, Amiri, Noto Naskh Arabic
- **تأثيرات زجاجية (Glassmorphism)** حديثة
- **أنماط إسلامية** زخرفية في الخلفية
- **تصميم متجاوب** مع جميع الأجهزة

### 🎬 الحركات والأنميشن
- **Three.js**: نظام جسيمات 3D تفاعلي
- **GSAP**: حركات متقدمة مع ScrollTrigger
- **تأثيرات 3D** على البطاقات (Tilt Effect)
- **مؤشر مخصص** للفأرة (Custom Cursor)
- **ظهور تدريجي** للعناصر عند التمرير

### 📱 الأقسام
1. **الرئيسية**: Hero Section مع تأثيرات ثلاثية الأبعاد
2. **عن المطعم**: قصة المطعم والمميزات
3. **المنيو**: تصفية حسب المطابخ العربية
4. **التجربة**: ليالي مدار والفعاليات الخاصة
5. **المعرض**: معرض صور تفاعلي
6. **الحجز**: نموذج حجز متكامل

### 🛡️ التقنيات المستخدمة
- **HTML5** - هيكل الصفحة
- **Tailwind CSS** - التصميم والتنسيق
- **JavaScript (Vanilla)** - التفاعل
- **Three.js** - الرسوميات ثلاثية الأبعاد
- **GSAP** - مكتبة الحركات المتقدمة
- **Lucide Icons** - الأيقونات

## 🚀 طريقة الاستخدام

### 1. تشغيل الموقع محلياً
```bash
# استخراج الملفات
unzip madar-restaurant.zip

# الدخول للمجلد
cd madar-restaurant

# تشغيل خادم محلي (مثال باستخدام Python)
python -m http.server 8000

# أو باستخدام Node.js
npx serve

# ثم افتح المتصفح على
http://localhost:8000
```

### 2. رفع الموقع على استضافة
- يمكن رفع الملفات مباشرة على أي استضافة ويب (Netlify, Vercel, GitHub Pages)
- لا يحتاج لقاعدة بيانات أو backend
- كل الملفات ثابتة (Static)

## 🔧 التخصيص

### تغيير الصور
استبدل روابط الصور في الـ HTML بروابط صورك الخاصة:
```html
<img src="path/to/your/image.jpg" alt="Description">
```

### تغيير الألوان
عدل متغيرات Tailwind في قسم `<script>` في ملف index.html:
```javascript
colors: {
    gold: {
        500: '#D4A532',  // غير هذا اللون
    }
}
```

### إضافة أطباق جديدة
أضف عناصر جديدة في قسم المنيو:
```html
<div class="menu-item glass rounded-2xl p-6" data-category="new-category">
    <!-- محتوى الطبق -->
</div>
```
